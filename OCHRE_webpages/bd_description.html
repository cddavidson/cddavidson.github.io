DIGS 30003: Data Management for the Humanities Class Project

For my final project for this class I helped Dr. Foy Scalf with Spell 51 of the Book of the Dead

This class project included making an import document that reflected the different spellings in various manuscripts

which was then uploaded into OCHRE as the content pool from which all the manuscripts could be reconstructed.

The words were subdivided into epigraphic units, the smallest constituent elements of the words.

These epigraphic units were then joined to create discourse units. For example, hr.t-ntr is composed of three epigraphic units as shown on the left,

but they are combined to make a compound word meaning "necropolis" in the discourse hierarchy shown on the right.

Properties were also added to show the condition of the epigraphic units on the manuscript.

For example, when epigraphic units were damaged, rubricated, or omitted/emended, these qualities were recorded with checked boxes as shown below.

The final step of the project was making high quality jpgs that could be linked from the resource hierarchy to the discourse hierarchy.

Although in this case the resource hierarchy stored images, other data such as 3D models, sound files, and even maps from ArcGIS can be stored here.

Dr. Scalf provided a JSESH file for each manuscript, and I divided it according to the discourse hierarchy with each line being resized to fit an artboard in Adobe Illustrator for consistency.

The resource hierarchy also includes images of the manuscripts themselves so that Egyptologists can view the hieratic signs.

Finally, each image was linked from the resource hierarchy to the appropriate place where it appears in the discourse hierarchy with the linked items pane on the right.
